# Relax Game Hub (relax-game.xyh.wiki)

A pure front-end React + Vite website providing guides and tools for Cookie Clicker and idle games.

- Tech: React 18 + TypeScript + Vite
- SEO-ready: meta tags, OG/Twitter, sitemap.xml, robots.txt
- Privacy-friendly: all tools run locally in the browser (no upload)

## Scripts

```bash
npm install
npm run dev
npm run build
```
